 10.08.2009 [08]
 ---------------
 1.   В   журнале   регистраций  вместо  операции  [Удалить] операция [В
 архив].   Для   просмотра      записей     перемещенных     в     архив
 {Операции}/    {Журнал регистраций(архив)}.
 2. В журнале регистраций добавлен поиск по Ф.И.О. и/или номеру комнаты.
 Если  номер  комнаты  не  выбран  (№ комнаты: пусто), то будет выполнен
 поиск  по  Ф.И.О.,  если  данные введены, если номер комнаты выбран, то
 будет  выполнен  поиск  по  номеру  комнаты, если Ф.И.О.: пусто, или по
 номеру  комнаты  и  Ф.И.О.,  если  данные  введены.  Для очистки поля №
 комнаты откройте список комнат и нажмите кнопку [Закрыть].
 3.  При  удалении  записи ([В архив]) в журнале регистрации выполняется
 автоматическая перенумерация (№ пп).

 01.07.2008 [05]
 ---------------
 1.  Изменен алгоритм контроля за занятостью номеров. Теперь учитывается
 что  окончание  срока проживания/бронирования может совпадать с началом
 срока бронирования номера.

 20.06.2008 [04]
 ---------------
 1. Добавлена печать журнала регистраций "как есть".

 03.06.2008 [03]
 ---------------
 1. Добавлено еще одно состояние номера 'Оплачен'.
 2.    При   выполнении контроля  за  свободным/занятым  номером  теперь
 учитывается,  что  заселение/бронирование  может  быть в день окончания
 срока проживания/бронирования.

 ---------------
 1. Номера комнат только цифра.

 приезда.
 3.  Добавлена  возможность  вывода  на  печать  данных  "Номерной фонд"
 (кнопка [Печать]).
